          九、成本管理

  • 成本控制￼的工具与技术
    • 挣值管理 EVM
      • AC 实际成本 Actual cost
      • EV 挣值 Earned value
      • PV 计划值 Planed value
      • SV 进度偏差￼ ::￼Schedule Variance::
        • EV - PV
      • CV 成本偏差
        • Cost Variance
        • EV - AC
      • SPI 进度绩效指标
        • Schedule Performance Index
        • EV / PV
      • CPI 成本绩效指数
        • Cost Performance Index
        • EV / AC
    • 预测
      • BAC 完工预算
        • Budget cost At Completion
      • EAC 完工估算
        • Estimate At Completion
        • AC + ETC
        • 典型偏差计算
          • BAC / CPI
      • ETC 完工尚需估算
        • Estimate To Complete
        • 非典型计算
          • BAC - EV
        • 典型偏差计算
          • ( BAC - EV ) / CPI
    • 完工尚需绩效指数
    • 绩效审查

      成本估算

  • 成本基准（BAC￼）￼不包括管理储备
  • 项目￼预算=成本基准+管理储备
  • 三点估算法（PERT) (1+4+1)/6
