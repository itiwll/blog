Babel 的 Class 的转换

ES6 加入了 Class 的语法，Class 的大部分功能 ES5 都可以做到，我们可以把 class 看作是一个语法糖。本篇文章我们一步步来看看 Babel 是怎么把 class 转换成 ES5 语法的。

观察方发

在 Babel 的官网上有一个 Try it out 的网页可以方便的看到编译后的代码。我们要看转换成 ES5 的语法，我们在 ENV PRESET 取消勾选 Enabled.
